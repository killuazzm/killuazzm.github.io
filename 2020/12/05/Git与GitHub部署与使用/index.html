生成ssh密匙

打开Git Bash，输入ssh-keygen -t rsa

生成的id_rsa与id_rsa.pub文件分别是私匙和公匙

将ssh添加至Github

进入setting–>SSH and GPG keys–>New SSH key

将文件id_rsa.pub的内容复制至key单元格中，单击Add SSH key添加

在Git中输入ssh -T git@github.com进行测试

提交代码

创建一个仓库

在仓库中找到ssh地址并复制

进入将要上传至GitHub的项目的文件夹中，使用git打开它

git init

git commit -m "first commit"

git remote add origin [你的仓库的ssh]
